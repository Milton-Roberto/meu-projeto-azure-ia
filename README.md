# Laboratório - Azure Speech Studio e Language Studio

Este repositório contém minha prática do laboratório proposto pela DIO, explorando ferramentas de IA para análise de fala e linguagem natural utilizando os serviços do Azure.

## Objetivo do Projeto
- Aplicar os conceitos aprendidos sobre IA generativa, reconhecimento de fala e processamento de linguagem natural.
- Registrar o processo realizado durante os testes no Speech Studio e Language Studio.
- Publicar a documentação em um repositório GitHub.

---

## Speech Studio

### Experimentos Realizados
1. **Transcrição de Áudio**
   - Eu carreguei um arquivo de áudio e o sistema converteu automaticamente em texto.
   - Resultado: A precisão foi boa, e o modelo identificou corretamente pausas e entonações.

2. **Conversão de Texto para Fala**
   - Testei a geração de fala com diferentes vozes (masculinas, femininas e neurais).
   - Resultado: As vozes neurais têm entonação muito mais natural.

---

## Language Studio

### Experimentos Realizados
1. **Análise de Sentimento**
   - O sistema identificou corretamente sentimentos positivos, negativos e neutros.

2. **Extração de Palavras-Chave**
   - Útil para entender rapidamente do que o texto trata.

3. **Reconhecimento de Entidades**
   - Identificou nomes de pessoas, lugares e organizações no texto analisado.

---

## Conclusão

Este laboratório me permitiu compreender na prática como serviços de IA podem ser integrados em soluções reais. As ferramentas são simples de usar e muito poderosas para automações, chatbots, análise de dados e acessibilidade.
